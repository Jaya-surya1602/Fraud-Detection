# 💳 Credit Card Fraud Detection

## 📌 Project Overview
This project detects fraudulent credit card transactions using Machine Learning techniques. It helps in identifying suspicious activities and preventing financial loss.

## 🚀 Features
- Detects fraud transactions
- Uses ML algorithms
- High accuracy prediction
- Handles imbalanced dataset

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## 📂 Files in this Repository
- `Credit Card Fraud Detection.ipynb` → Model building
- `credit_card_model.pkl` → Saved ML model
- `README.md` → Project documentation

## 📊 Dataset
- Credit card transaction dataset (imbalanced dataset)

## ▶️ How to Run
1. Download the repository
2. Open Jupyter Notebook
3. Run the `.ipynb` file

## 📈 Output
- Predicts whether transaction is Fraud or Not
